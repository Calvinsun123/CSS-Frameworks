CSS Frameworks

CSS framework is a collection of a series of CSS files, including basic element reset, page layout, grid layout, form style, general rules and other code blocks, used to simplify the work of web front-end development and improve work efficiency feature

a) format reset.css The real benefit of formatting css is to get things started quickly, you can introduce frames in new HTML files without having to deal with resetting the pc) Basic styles type.css Define font size and color for body, h1-h6, a:link-a:active, p, etc. adding and margin to achieve unified layout and the same performance in browsers. b) layout layout.css The design of a website may have many kinds of layouts, but most of them are composed of several reusable layouts. By selectively introducing the required layouts, the desired page layout can be quickly applied. d) table decoration table.css Define the performance of tags such as table, tr, td, th, thead, tfoot, tbody, caption, etc. e) Form decoration form.css Define the performance of fieldset, label, button, input, select, and textarea.
